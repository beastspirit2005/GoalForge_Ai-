# GoalForge AI — Presentation / Demo Notes

## Elevator Pitch (30 seconds)
"GoalForge AI isn't just a goal tracker — it's an intelligent performance coach. Employees create goals, our AI generates weekly milestones and risk assessments using Google Gemini, managers get early warning signals, and admins see organization-wide execution health. We turn goal-setting from a quarterly checkbox into a continuous, AI-guided performance improvement loop."

## Demo Flow (5 minutes)

### 1. Landing Page (30s)
- Show the landing page with key features
- Highlight: AI milestone planning, risk prediction, role dashboards

- Next.js 16 + FastAPI + PostgreSQL + Google Gemini

## Key Talking Points

1. **Not just tracking — coaching**: AI actively helps achieve goals
2. **Adaptive**: Recommendations change as progress changes
3. **Three perspectives**: Employee plans → Manager reviews → Admin monitors
4. **Real architecture**: PostgreSQL, JWT auth, proper API design
5. **AI with fallback**: Works without Gemini API key (graceful degradation)
6. **Governance built-in**: Full audit trail of every change

## Potential Questions & Answers

**Q: What happens without the Gemini API key?**
A: The system falls back to template-based milestones and heuristic risk scoring. The full product works without AI.

**Q: How does risk scoring work?**
A: It combines progress percentage, days until deadline, and workload (number of active goals). AI adds contextual analysis when available.

**Q: What validation rules exist?**
A: Max 8 goals per employee, minimum 10% weightage per goal, total weightage must equal 100%.

**Q: Is the auth real?**
A: Yes — JWT tokens with bcrypt password hashing. The demo also has a quick role-switcher for presentation convenience.
